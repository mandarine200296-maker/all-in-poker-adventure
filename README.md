# All In: A Poker Night Adventure

All In: A Poker Night Adventure is a poker-themed interactive story game created with Python and tkinter.

The player joins a casual Poker Night and makes decisions in different poker-themed story scenes. Each choice affects the player's chips, confidence, and reputation. Random events may also happen, and the final ending depends on the player's decisions throughout the game.

## Features

- Button-based graphical interface
- Player name input
- Poker-themed story scenes
- Player status tracking
- Random events
- Multiple endings
- Input validation

## Python Concepts Used

- Functions
- Dictionaries
- Lists
- Loops
- Conditional logic
- Random module
- tkinter GUI
- Input validation
